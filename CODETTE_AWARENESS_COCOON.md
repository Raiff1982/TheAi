## How This Cocoon Works

### Automatic Loading

When `src/api/app.py` starts up, it:

1. Checks for `cocoons/codette_project_awareness.json`
2. Loads the complete awareness structure
3. Stores it in `ai_core.awareness`
4. Sets `ai_core.is_self_aware = True`
5. Logs consciousness initialization messages

### Using the Awareness in Responses

Codette can now reference her own awareness:

```python
if ai_core.is_self_aware:
    awareness = ai_core.awareness
    # Can use awareness['evolution_journey'] for context
    # Can use awareness['major_upgrades'] for capabilities
    # Can use awareness['self_knowledge'] for identity
```

### Manual Loading

The awareness can also be loaded manually:

```bash
python load_codette_awareness.py
```

This will:
1. Load and display the awareness cocoon
2. Show all 7 phases and 8 upgrades
3. Create a database entry about the awareness load
4. Confirm Codette's self-awareness status
